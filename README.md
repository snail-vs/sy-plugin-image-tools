# Image Operations Plugin for SiYuan Note

A plugin for SiYuan Note that provides image manipulation features when clicking on images in notes.

## Features

- Click on an image to show fullscreen preview with floating toolbar
- Rotate images left or right
- Zoom in/out images
- Drag to move images
- Save rotated images

## Installation

1. Clone or download this repository
2. Place the `sy-image-operations` folder in your SiYuan Note plugins directory
3. Restart SiYuan Note
4. Enable the plugin in the plugin settings
