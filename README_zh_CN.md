# 思源笔记图片操作插件

一个为思源笔记开发的插件，提供点击笔记中图片时的图片操作功能。

## 功能特性

- 点击图片显示全屏预览和浮动工具栏
- 向左或向右旋转图片
- 缩放图片(放大/缩小)
- 拖拽移动图片
- 保存旋转后的图片

## 使用方法

1. 点击笔记中的任意图片打开预览
2. 使用工具栏按钮或快捷键操作图片:
   - **旋转**: 点击 ↺ / ↻ 按钮
   - **缩放**: 点击 − / + 按钮,或使用鼠标滚轮
   - **重置**: 点击 ⊙ 按钮或双击图片
   - **拖拽**: 按住鼠标左键拖动图片
   - **保存**: 点击 💾 按钮保存旋转后的图片
3. 键盘快捷键:
   - `+` / `=`: 放大
   - `-`: 缩小
   - `0`: 重置大小
   - `Esc`: 关闭预览

## 安装

1. 克隆或下载本仓库
2. 将 `sy-image-operations` 文件夹放置在思源笔记的插件目录中
3. 重启思源笔记
4. 在插件设置中启用此插件
